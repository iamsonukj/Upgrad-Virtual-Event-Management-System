📌 Upgrad Virtual Event Management System

The Upgrad Virtual Event Management System is a web-based application that allows users to explore, register, and manage virtual events. The system is built using HTML, CSS, Bootstrap, and JavaScript and demonstrates front-end development concepts such as client-side validation, DOM manipulation, and dynamic data handling using LocalStorage.

This application has two types of users:

Participant – Can view events and register.

Admin – Can log in and manage events.

🚀 Features
👤 Participant Features

View upcoming events on the Home Page

Register for events

Submit queries using the Contact Us page

🔑 Admin Features

Secure login using predefined credentials

Add new events

Search events by ID, Name, or Category

Delete events

Logout from the system
